# autismus-voismus
Бот для тех, кому не терпимы голосовые сообщения.

Использует библиотеку pyTelegramBotAPI (pip install pyTelegramBotAPI), рассчитан для запуска из под PyCharm или прочих IDE для работы с Python 3.10+.
